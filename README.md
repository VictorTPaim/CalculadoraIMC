# Calculadora IMC

Aplicação console para cálculo do IMC.

## Funcionalidades

- Calcula o IMC
- Mostra em que faixa de IMC o usuário está.
- Exibe quanto peso falta para chegar ao IMC ideal.

## Tecnologias utilizadas

- dotnet 2.1

## Como executar

- Realize o clone do repositório e execute o comando: 

```
dotnet run
```
